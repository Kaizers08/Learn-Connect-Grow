┌──────────────────────────────────────────────────────────────────┐
│  HOW TO USE                                                      │
└──────────────────────────────────────────────────────────────────┘

  AS MENTOR:
  ──────────
  1. Login → Dashboard
  2. Click "Learning Materials" (sidebar)
  3. Click "➕ Upload Material"
  4. Fill form + select file
  5. Upload!
  6. Monitor mentees sa "Progress" section


  AS MENTEE:
  ──────────
  1. Login → Dashboard
  2. Click "Learning Materials" (sidebar)
  3. View materials
  4. Click "View" to see file
  5. Click "Mark Complete" ✅ when done
  6. Track progress sa "Your Progress" card


┌──────────────────────────────────────────────────────────────────┐
│  FEATURES                                                        │
└──────────────────────────────────────────────────────────────────┘

  ✅ Upload files (videos, PDFs, documents)
  ✅ Organized numbering (1.1, 1.2, 2.1, etc.)
  ✅ Progress tracking with checkboxes
  ✅ Progress bars (color-coded)
  ✅ File size limit: 50MB (free tier)
  ✅ Beautiful card layouts
  ✅ Mobile responsive
